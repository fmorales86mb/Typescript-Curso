Typescript es un lenguaje orientado a objetos con base de js
Es compilado, al compilarlo crea un archivo js que es el que se ejecuta y el que pueden leer los navegadores.

Para descargar ts: 
    node i tsc -g

para compilar un archivo ts: 
    tsc archivo.ts

para compilación automática: 
    tsc archiv.ts -w

para crear archivo de configuración:
    tsc -init

Let:
let permite declarar variables limitando su alcance (scope) al bloque, declaración, o expresión 
donde se está usando. Lo anterior diferencia  let de la palabra reservada var , la cual define una 
variable global o local en una función sin importar el ámbito del bloque.
